# Change Log
All notable changes to this project will be documented in this file. This file
follows the formatting recommendations from [Keep a
CHANGELOG](http://keepachangelog.com/). This project adheres to [Semantic
Versioning](http://semver.org/).

## [0.1.1] - 2015-3-30
This release includes more dummy commits than what I would have liked
because I needed to tweak travis builds.
### Fixed
- Support for ghc-7.6.3 and ghc-7.10.1.

## [0.1] - 2015-3-30
### Added
- Basic routing.
- HTTP parameters handling for both *query string* and form encoded `POST`s.
- Trailing slash redirection.
